# Retirement Chengdu Portal

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:5173

## 主要功能

- **Home页面**: 显示今日WFH/请假人员、WFH安排表、未来一周请假日历
- **Emailer页面**: 四种邮件模板生成器（OT启动/结束、WFH申请、请假申请）
